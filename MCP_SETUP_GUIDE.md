# Руководство по запуску MCP Web Search Server

## Быстрый старт (Windows)

### Способ 1: Автоматическая установка
Просто дважды кликните на файл `setup-mcp-server.bat` в корне проекта.

### Способ 2: Ручная установка

1. **Откройте терминал** (cmd или PowerShell) в папке проекта

2. **Перейдите в директорию сервера:**
   ```bash
   cd mcp-web-search-server
   ```

3. **Установите зависимости:**
   ```bash
   npm install
   ```

4. **Скомпилируйте TypeScript:**
   ```bash
   npm run build
   ```

5. **Перезапустите VS Code**

---

## Проверка установки

После перезапуска VS Code, MCP сервер будет автоматически доступен для Roo Code.

Конфигурация находится в файле `.roo/mcp.json`:
```json
{
  "mcpServers": {
    "web-search": {
      "command": "node",
      "args": ["./mcp-web-search-server/build/index.js"],
      "disabled": false,
      "alwaysAllow": [
        "web_search",
        "fetch_url",
        "get_current_time",
        "get_weather",
        "get_news",
        "get_exchange_rate"
      ]
    }
  }
}
```

## Доступные инструменты

После настройки AI-ассистент получит доступ к следующим инструментам:

| Инструмент | Описание | Пример использования |
|------------|----------|---------------------|
| `web_search` | Поиск в интернете | "Найди последние новости про AI" |
| `fetch_url` | Получение содержимого веб-страницы | "Получи содержимое с сайта example.com" |
| `get_weather` | Погода в любом городе | "Какая погода в Москве?" |
| `get_news` | Последние новости | "Покажи последние новости технологий" |
| `get_current_time` | Текущее время в любой зоне | "Сколько сейчас времени в Нью-Йорке?" |
| `get_exchange_rate` | Курсы валют | "Курс доллара к рублю" |

## Решение проблем

### Сервер не запускается
1. Убедитесь, что Node.js установлен: `node --version`
2. Проверьте, что сборка прошла успешно: файл `mcp-web-search-server/build/index.js` должен существовать
3. Перезапустите VS Code полностью

### Инструменты не появляются
1. Проверьте файл `.roo/mcp.json` - он должен быть в корне проекта
2. Убедитесь, что путь к `build/index.js` корректный
3. Откройте настройки Roo Code и проверьте раздел MCP Servers

### Ошибки при поиске
- Некоторые сайты могут блокировать автоматические запросы
- Попробуйте другой поисковый запрос
- Проверьте подключение к интернету

## Дополнительная информация

- MCP сервер работает через stdio транспорт
- Не требует API ключей - использует бесплатные публичные API
- Автоматически обрабатывает CORS через прокси-серверы
- Поддерживает извлечение чистого текста из HTML
